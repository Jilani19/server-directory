# DATA SOURCE MATRIX

| Target Domain | Supported Primary Source | Fallback Database / Integration |
| :--- | :--- | :--- |
| **Financial Intelligence** | SEC EDGAR Filings (10-K, 10-Q) | OpenCorporates |
| **Clinical Intelligence** | ClinicalTrials.gov (NCT IDs) | WHO ICTRP |
| **Product Intelligence** | FDA Orange Book / OpenFDA | EMA EPARs |
| **Regulatory Intelligence** | FDA Warning Letters | CDSCO / MHRA Alerts |
| **Scientific Output** | PubMed (NCBI) | Crossref (DOIs) |
| **Intellectual Property** | USPTO Patent Grants | WIPO |
| **Corporate Identity** | GLEIF (LEI codes) | EDGAR (CIK codes) |
